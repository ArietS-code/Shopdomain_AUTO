# store/ README

This folder documents Vuex store setup for the workshop UI. It introduces the root store and the cart module.

Contents:
- `cart-store.prompt.md` – specification of cart state, mutations, actions, getters.
- `cart-store.example.md` – implementation snapshot of `src/store/cart.ts`.

Root store file: `src/store/index.ts` mounts `cart` module and exports the store.
