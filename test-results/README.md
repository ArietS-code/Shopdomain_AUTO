# Test Results Directory

This directory contains all test execution artifacts generated by Playwright.

## Structure

```
test-results/
├── screenshots/          # Timestamped screenshots (persisted)
├── html-report/         # HTML test reports (auto-generated)
├── artifacts/           # Videos, traces (auto-generated)
└── .gitkeep files       # Preserve directory structure
```

## Output Files

### Screenshots (`screenshots/`)
- **Format**: `{testname}_{timestamp}.png`
- **When**: Taken on test failure or manually via `takeScreenshot()`
- **Retention**: Persisted across test runs
- **Full page**: Yes

### HTML Reports (`html-report/`)
- **Generated**: After each test run
- **Auto-open**: Yes (configured in playwright.config.ts)
- **Manual view**: `npm run test:e2e:report`
- **Contents**: Test results, timings, screenshots, traces

### Artifacts (`artifacts/`)
- **Videos**: Recorded on test failure
- **Traces**: Debug traces for failed tests
- **Retention**: Only on failure
- **Cleanup**: Auto-cleared on next run

## Configuration

Settings defined in `playwright.config.ts`:
```typescript
outputDir: 'test-results/artifacts'
screenshot: 'only-on-failure'
video: 'retain-on-failure'
trace: 'on-first-retry'
```

## Viewing Results

### HTML Report
```bash
npm run test:e2e:report
```

### Individual Artifacts
Navigate to:
- Screenshots: `test-results/screenshots/`
- Videos: `test-results/artifacts/*.webm`
- Traces: Open with `npx playwright show-trace test-results/artifacts/trace.zip`

## Cleanup

```bash
# Remove all test results except screenshots
rm -rf test-results/html-report test-results/artifacts

# Remove everything including screenshots
rm -rf test-results
mkdir -p test-results/screenshots
```

## Git Tracking

- ✅ **Tracked**: Directory structure (`.gitkeep` files)
- ❌ **Ignored**: All generated content (screenshots, reports, artifacts)
